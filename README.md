# 📈 Crypto Data Pipeline & Interactive Dashboard

Това е Python приложение, което изгражда автоматизиран конвейер за данни (Data Pipeline) за следене и визуализация на цените на криптовалути в реално време. Проектът демонстрира умения за работа с функционално програмиране, бази от данни (CSV), обработка на данни и изграждане на уеб интерфейси.

## 🚀 Архитектура на проекта
Проектът е разработен изцяло чрез **функционален подход** (без излишен ООП баланс), което прави компонентите независими, леки и лесни за тестване:
* `scraper.py` — Модул за симулирано извличане на пазарни данни с отчитане на времеви маркери. Включва `try-except` блокове за дефанзивно програмиране и защита от сривове.
* `analytics.py` — Модул, захранван от библиотеката **Pandas**, който превръща суровите данни в DataFrame структури и поддържа динамична история в CSV формат.
* `App.py` — Интерактивно уеб табло (Dashboard), изградено със **Streamlit**, което визуализира данните чрез динамични графики и метрики в реално време.

## 🛠️ Технологичен стек
* **Python 3.14** (използващ вградени библиотеки `random` и `datetime`)
* **Pandas** (за управление, трансформация и дописване на таблични данни)
* **Streamlit** (за изграждане на реактивен потребителски уеб интерфейс)

## 📦 Как се стартира локално
1. Клонирайте проекта на компютъра си.
2. Инсталирайте нужните пакети:
   ```bash
   pip install pandas streamlit
   ```
3. Стартирайте уеб приложението чрез командата:
   ```bash
   python -m streamlit run App.py
   ```
