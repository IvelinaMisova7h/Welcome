# 📈 Crypto Data Pipeline (Биткойн Симулатор)

Това е Python приложение, което симулира автоматизиран конвейер за данни (Data Pipeline) за следене на цените на криптовалути в реално време. Проектът демонстрира умения за работа с функционално програмиране, обработка на данни и изграждане на софтуерна архитектура.

## 🚀 Архитектура на проекта
Проектът е изграден изцяло чрез **функционален подход** (без излишен ООП баланс), което прави кода лек, бърз и лесен за тестване:
* `scraper.py` — Модул за симулирано извличане на пазарни данни с отчитане на времеви маркери. Включва `try-except` блокове за дефанзивно програмиране и защита от сривове.
* `analytics.py` — Модул, захранван от библиотеката **Pandas**, който превръща суровите данни в DataFrame структури и поддържа динамична история в CSV формат.

## 🛠️ Технологичен стек
* **Python 3.14** (използващ библиотеки `random` и `datetime`)
* **Pandas** (за управление и дописване на таблични данни)

## 📦 Как се стартира локално
1. Клонирайте проекта на компютъра си.
2. Инсталирайте нужните пакети: `pip install pandas`
3. Стартирайте събирането на данни чрез: `python analytics.py`
